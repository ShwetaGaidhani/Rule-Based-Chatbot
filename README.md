# Rule-Based Chatbot

This project is a simple rule-based chatbot developed using Python and pattern matching. The chatbot can recognize user inputs, respond using predefined rules, and answer basic technology-related questions using a small knowledge base.

This project was created as part of an Artificial Intelligence internship task.

## Features

* Pattern matching chatbot
* Supports intents like greeting, help, and small talk
* Basic technology knowledge base
* Interactive console conversation
* Conversation history tracking

## Topics Supported

The chatbot can answer questions about:
Python, JavaScript, Java, C++, Ruby, Artificial Intelligence, Machine Learning, API, Database, Git, Cloud Computing, Algorithm, and React.

Example questions:

* What is Python?
* Tell me about AI
* Explain Machine Learning

## Note

Conversation history files such as `chat_history.txt` and `chat_history.json` are generated automatically when the chatbot runs.
